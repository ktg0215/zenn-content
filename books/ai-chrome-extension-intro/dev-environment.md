## 使うツールはたった3つ

難しいことは何もありません。使うのは以下の3つだけです。

| ツール | 役割 | 月額コスト |
|--------|------|----------|
| **Cursor** または **VS Code** | コードエディタ（AI補完あり） | 無料〜$20 |
| **Claude** または **Claude Code** | コード生成・デバッグ補助 | 無料〜$20 |
| **Node.js + npm** | ビルドツールの実行環境 | 無料 |

合計コスト：最安で無料、Claude Proを使っても月$20程度です。Chrome Web Storeの登録料は初回$5（一回限り）のみ。

### AIツールの使い分け

| AI | 得意なこと |
|---|-----------|
| Claude | 設計、長文指示書、コード生成、デバッグ |
| ChatGPT | アイデア出し、一般的な質問 |
| Gemini | 最新情報のリサーチ |

設計と実装はClaudeに集中投資するのがおすすめです。Claudeだけ有料版にして、他は無料で補完する構成が費用対効果が高い。

## セットアップ手順

### Step 1：Node.jsのインストール

Node.jsはJavaScript/TypeScriptのビルド環境です。公式サイト（nodejs.org）からLTS版をダウンロードしてインストールします。

```bash
# インストール後、バージョン確認
node -v  # v20以上を確認
npm -v   # 正常にインストールされているか確認
```

### Step 2：ビルドツールの選択

Chrome拡張のビルドツールには主に3つの選択肢があります。

| ツール | 特徴 | おすすめ対象 |
|--------|------|------------|
| **WXT** | フレームワーク型、設定が少ない | 初心者・React使いたい人 |
| **Vite + CRXJS** | 柔軟、カスタマイズしやすい | 中級者 |
| **Vanilla JS** | 設定不要、シンプル | 最小構成のみ必要な場合 |

本書では **WXT** を使います。設定ファイルが少なく、TypeScript + Reactがすぐに使えるためです。

```bash
# WXTプロジェクトを新規作成
npx wxt@latest init my-extension
cd my-extension
npm install
npm run dev  # 開発サーバー起動（Chrome上でリアルタイム反映）
```

### Step 3：Cursorでプロジェクトを開く

Cursorを開いてプロジェクトフォルダを選択します。Cursor AIに指示するだけでコードが自動生成されます。

```
# entrypoints/popup/App.tsx
# Cursor AIへの指示例（Cmd+K または Chat欄に入力）
「このChrome拡張に、現在のページのタイトルと
URLをコピーするボタンをポップアップに追加してください。
TypeScript + Reactで実装してください。」
```

## AIへのコード依頼のコツ

18本の拡張を作ってわかったことは、**AIへの指示の質が出力コードの質を決める**ということです。

### 良い指示の3要素

**① Chrome拡張であることを明示**
「Chrome拡張（Manifest V3）で〜」と最初に書く。これがないとAIがWebアプリ用のコードを生成することがある。

**② 技術スタックを指定**
「TypeScript + React + WXTフレームワークで〜」と明示する。スタック指定がないとAIが毎回違う構成を選ぶ。

**③ 完成形を具体的に描く**
「ポップアップUIに〇〇ボタンがあり、クリックすると〇〇する。データはchrome.storage.localに保存する」まで書く。

### エラーが出たときの対処

```
# Cursorのチャット欄にそのまま貼る
「以下のエラーが出ています。修正してください：

[エラーメッセージをそのままコピペ]

ファイル: entrypoints/popup/App.tsx」
```

ほとんどのエラーはAIが即座に解決策を提示します。「なぜエラーが出たか」を理解できなくても、まず動かすことを優先しましょう。

## よくある最初の壁と解決策

| 症状 | 原因 | 解決策 |
|------|------|--------|
| ビルドが通らない | Node.jsのバージョンが古い | `node -v` でv20以上を確認、更新 |
| 拡張がChromeで動かない | dist/を読み込んでいない | `chrome://extensions` でデベロッパーモードON |
| コードの意味がわからない | — | AIに「このコードを日本語で説明して」と聞く |
| Cursorが遅い | — | Claude Codeをターミナルから直接使う |
