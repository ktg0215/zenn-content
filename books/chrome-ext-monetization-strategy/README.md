### 案A：「収益グラフ」上昇型

**テーマ**: 数字が上がる。投資する価値を視覚化。

```
背景: #0D1117（ほぼ黒）
アクセント: #22C55E（グリーン）
レイアウト:
  - 上部: 右肩上がりのラインチャート（細い緑線＋終点ドット）
  - 中央: タイトルテキスト（白、大）
  - 下部: サブタイトル「18本のデータから蒸留した収益化の全手順」（グレー）
  - 右下隅: ¥500バッジ（小さく、緑背景）
フォント: Geist Mono（チャート数値） / Noto Sans JP（タイトル）
```

**キャッチ**: 「インストール数は伸びているのに、収益が出ない」に答える本

---

### 案B：「実データ比較表」ドキュメント型

**テーマ**: 実務書感。表やコードが並ぶことで「すぐ使える」を伝える。

```
背景: #FAFAFA（オフホワイト）
アクセント: #1A1A1A（黒）
レイアウト:
  - 全体: 薄いグリッド線（方眼紙イメージ）
  - 中央上部: 小さい比較表スニペット（340%増、転換率1.9%→3.2%等の数値が並ぶ）
  - 中央: タイトルテキスト（黒、大）
  - 下部: 著者名 + ¥500
フォント: Geist Sans（タイトル）/ Geist Mono（数値スニペット）
```

**キャッチ**: 「再現性のある収益化」を設計図っぽく表現

### 案C：「拡張アイコン × 円マーク」変換型（推奨）

**テーマ**: 拡張をお金に変える過程を視覚化。Book 2の核心メッセージ。

```

アクセント: #F59E0B（アンバー / ゴールド）

  - 上部中央: Chromeパズルアイコン（グレー）→ 矢印（→）→ ¥マーク（ゴールド）
```
