## カバー画像 3案

サイズ：1000×1400px（Zenn Book標準）/ 形式：PNG / 作成ツール：Canva

Book 1〜3（ダーク×テック系）と対比して、Book 4はエッセイ感・温かさを重視。

---

### 案A：「光の窓」ライフスタイル型（推奨）

**テーマ**: 個人開発は「技術」ではなく「ライフスタイル」というメッセージ。

```
背景: #FAFAF8（オフホワイト）または #F5F0E8（クリーム）
アクセント: #E07B39（テラコッタオレンジ）
レイアウト:
  - 中央〜上: 窓から光が差し込むシンプルなイラスト（線画）
             デスクにノートPC、コーヒーカップ
  - 中央: タイトルテキスト（ダークグレー、太め）
  - 下部: サブタイトル + 「無料公開」バッジ
フォント: Noto Serif JP（手書き感・読み物感）
```

**キャッチ**: 「続けるための本」という温かさをビジュアルで表現

---

### 案B：「手書き風テキスト」エッセイ型

**テーマ**: 技術書ではなく日記・手記というニュアンス。

```
背景: #FFFEF7（ページ風クリーム）
アクセント: #3B82F6（インクブルー）
レイアウト:
  - 全体: 罫線ノート風のテクスチャ（薄い横線）
  - 上部: 小さなペンのアイコン or 手書き風装飾
  - 中央: タイトルテキスト（手書きフォント or セリフ体）
  - 下部: 「18本の実体験から」サブコピー + 著者名
フォント: Noto Serif JP + Geist Mono（数字部分）
```

**キャッチ**: 「日記を読む」感覚で読めるエッセイ

---

### 案C：「数字と余白」ミニマル型

**テーマ**: 「18本」という実績数字を前面に出す。シンプルで潔い。

```
背景: #FFFFFF（白）
アクセント: #111827（ダークグレー）+ #EF4444（レッド、"18"だけ）
レイアウト:
  - 上部: 大きく "18" （赤、Book 1〜3と対比して目立つ）
  - 中央: タイトルテキスト（ダーク、大）
  - 下部: 「── 無料公開」のみ
フォント: Geist Sans（タイトル）/ Noto Sans JP（サブ）
```

**キャッチ**: 「18本という実績の説得力」をビジュアルで先出し

---

**推奨**: 案Aが Book 4の「やわらかい」エッセイ感を最もよく表現。Book 1〜3のダーク×テック系と並べたとき、4冊の中で「読み物」として差別化される。オフホワイト×テラコッタは温かさとクリエイティブ感を両立。
